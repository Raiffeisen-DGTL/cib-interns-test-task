# DemoAPISocks
Приложение для автоматизации учёта носков на складе магазина.
Имеет следующие возможности:

приход и отпуск носков;
общее количество носков определенного цвета и состава в данный момент времени.

# Хранения данных системы
В качестве базы данных использует PostgreSQL

Для создания образа используются: Dockerfile и init.sql при помощи команды docker build . -t sock-docker

База запускается в отдельном докер контейнере при помощи команды

docker run --name sock -p 5432:5432 -d sock-docker

Таблицу приложение создает автоматически и заполняет ее данными

# Запросы

http://localhost:8080/api/socks/income - регистрирует приход носков на склад.
http://localhost:8080/api/socks/outcome - регистрирует отпуск носков со склада.

Тело POST-запроса выглядит в обоих случаях так:

{
    "color": "red",
    "cottonPart": 42,
    "quantity": 1
}

http://localhost:8080/api/socks?color=white&cottonPart=10&operation=moreThan - Возвращает общее количество носков на складе, соответствующих переданным в параметрах критериям запроса.
