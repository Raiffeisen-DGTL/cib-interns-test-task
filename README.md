#Raiffeisen test task

###Особенности:
- Приложение использует связку Spring + postgres;
- Необходимо поднять в докере бд через docker-compose.yaml, лежит в корне;
- По умолчанию БД использует порт 5433, приложение - 8099;
- При запуске приложения инициализируются таблицы и заполняются демонтстрационными данными; 
- Для удобства добавлен swagger UI: http://localhost:8099/api/swagger-ui.html.
