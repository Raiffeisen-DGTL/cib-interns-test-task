# Приложение для автоматизации учёта носков

## Стек технологий
* Java 11
* Gradle
* Spring Boot
* PostgreSQL
* Docker
* Flyway
* Swagger
* Heroku

## Запуск приложения
Локально приложение можно запустить командой `docker-compose up`.

Также сервис развернут на `Heroku` по адресу [https://socks-service-97631.herokuapp.com/](https://socks-service-97631.herokuapp.com/).
Описание всех методов сервиса доступно в `Swagger`'е по адресу
[https://socks-service-97631.herokuapp.com/swagger-ui/](https://socks-service-97631.herokuapp.com/swagger-ui/).
