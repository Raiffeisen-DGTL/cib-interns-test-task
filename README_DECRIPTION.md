# README #

Socks Store Project

### Properties and Tools ###

* Framework to configure an app - Spring Boot
* Database Server type - PostgreSQL
* Database versioning is implemented by Liquibase
* App and database are deployed in two Docker containers; bind with Docker compose
* Combined containers are deployed on AWS
* Endpoints persistence with Swagger UI is available at: [socks-store](http://ec2-3-145-3-43.us-east-2.compute.amazonaws.com:8090/swagger-ui/#/socks-controller)
